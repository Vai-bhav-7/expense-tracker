# Expense Management System using MERN Stack

## Project Description

The Expense Management System is a web-based application to help users track and manage their daily expenses. Built using the MERN stack (MongoDB, Express.js, React.js, Node.js), it is scalable, responsive, and user-friendly.

## Features

- **User Authentication:** Secure signup and login with role-based access control.
- **Expense Management:** Create, update, and delete expense entries and categories.
- **Dashboard & Reporting:** Visualize expenses with charts and summaries.
- **Responsive UI:** Works across desktop, tablet, and mobile devices.

## Tech Stack

- **Frontend:** React, Redux, react-bootstrap, Material Icons, tsparticles  
- **Backend:** Node.js, Express.js  
- **Database:** MongoDB
